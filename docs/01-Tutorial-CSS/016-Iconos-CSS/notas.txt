Fuentes en CSS

Trabajando en la elaboración del Tutorial e investigación detallada de la implementación de iconos con CSS. 
Según la famosa página de tutoriales “w3schools.com” tenemos la alternativa de incorporar iconos usando enlaces externos a servicios como Font Awesome,Bootstrap y Google Icons. Claro existen otras alternativas Como por ejemplo descargas directas de ficheros con extensión SVG. Sin embargo esto no queda aquí, también tenemos la posibilidad de aprender las técnicas que utilizan estas compañías. Aunque para el desarrollo costo-efectivo, esta alternativa puede pasar por alto, sin descartar que debemos tener en cuenta como es su creación y funcionamiento interno. Ya que debemos estar preparados para crear todo tipo de aplicaciones. Si que en mí próximo tutorial mostrare las alternativas para crear iconos y su implementación dentro de nuestras aplicaciones web.
